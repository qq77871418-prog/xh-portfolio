FGM电竞招募海报

Poster Design / Event Visual

以电竞元素与力量感视觉营造热血氛围，突出战队精神，吸引玩家关注与加入。

项目类型：海报设计项目编号：05工具：Figma / Photoshop

项目背景

FGM电竞战队面向核心玩家与潜在新成员发起招募，需要一张具备强烈视觉冲击力与辨识度的宣传海报，在社群与社交平台中快速传播。

设计目标

传递电竞的热血与战斗精神，突出战队品牌与招募信息，吸引目标玩家关注并加入。

受众定位

16-30岁热爱射击类游戏，追求竞技、团队、荣誉与个性表达的玩家。

视觉关键词

电竞 / 招募 / 热血 / 战斗 / 紫黑 / 闪电 / 力量 / 未来感

设计思路与策略

DESIGN STRATEGY

01 强力标题

采用大面积斜切字体与点阵纹理，强化信息层级。

02 紫黑配色

紫黑基调搭配荧光紫粉，营造神秘热血氛围。

03 闪电纹理

运用闪电、喷溅、裂痕元素，增强画面张力。

04 战斗氛围

角色与力量元素组合，传递战斗与胜利象征。

05 信息强化

通过斜向构图与箭头元素引导视线，突出招募信息。

项目总结

本次招募海报以强烈的视觉冲击与电竞文化为核心，结合闪电与战斗元素，成功塑造了战队的硬核形象与热血氛围。
